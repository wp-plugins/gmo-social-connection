== Description ==

This plugin inserts social share links at the top or bottom of each post.

This is the list of used social sites:

*  Facebook
*  Facebook
*  Twitter
*  Google+

== Installation ==

1. Upload this directory to the 'wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open 'GMO Social Connection Options' control panel through the 'Settings' menu and change configurations (This step is not required, the plugin has default settings)
